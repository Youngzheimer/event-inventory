# PackTrack - 행사 재고관리

행사 물품(XLR 케이블, 마이크 등)을 관리하고, 상자별 Data Matrix 코드로 현장에서 빠르게 재고를 확인하는 PWA입니다.

## 실행

```bash
cd event-inventory
npm install
npm run dev
```

아이패드/아이폰에서 PWA로 설치하려면 HTTPS 환경에서 접속 후 "홈 화면에 추가"를 선택하세요.

## 주요 기능

- **행사별 물품 관리** — 수량, 메모, 태그(기관/개인/대여) 지원
- **커스텀 체크 단계** — 있는지 확인, 적재, 현장 확인 등 단계 추가/삭제
- **태그별 체크 제외** — 특정 태그 물품을 특정 체크 단계에서 제외
- **부족 기록** — 없을 때 수량·사유 기록
- **상자/적재함 + Data Matrix** — QR 대신 인식률 높은 Data Matrix 라벨 인쇄
- **스캔 → 상자 내용** — 카메라로 스캔하면 해당 상자 물품·체크박스 즉시 표시
- **오프라인** — IndexedDB 저장, PWA로 설치 시 오프라인 사용 가능

## 사용 흐름

1. 새 행사 생성
2. 설정에서 체크 단계·태그 확인/수정
3. 상자 생성 → 라벨 인쇄 → 상자에 부착
4. 물품 추가 시 상자 지정
5. 현장에서 스캔 탭 → Data Matrix 스캔 → 체크 진행
